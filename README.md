# Agricultura Chile - NewCooltura Informada

Modulo de informacion para agricultores y el sector agropecuario. Parte del ecosistema NewCool.

## Funcionalidades

- **Buscador INDAP**: 16 oficinas regionales con filtros
- **Programas de Apoyo**: PRODESAL, PDTI, Creditos, Riego, Alianzas, SIRSD-S
- **Calculadora de Rentabilidad**: Calcula utilidad por hectarea
- **Tipos de Cultivo**: Hortalizas, frutales, cereales, ganaderia, apicultura, forestal
- **Glosario Agricola**: Terminos tecnicos explicados

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion

## Desarrollo

```bash
npm install
npm run dev
```

## Datos

Informacion basada en datos oficiales de INDAP (indap.gob.cl).
